CODEX Embeddable Widget

Drop scripture into any HTML page in one line. The <codex-reader> custom element renders a verse or passage with optional commentary, in your site's color scheme.

Quick start

<script src="https://codex.app/widget.js"></script>
<codex-reader book="john" chapter="3" verse="16"></codex-reader>

Attributes

  • book — required. Book id (e.g. john, gen, ps, rom, rev).
  • chapter — required. Chapter number.
  • verse — optional. Single verse, range (5-10), or list (3,7,12). Omit for whole chapter.
  • translation — optional. Default web. Any translation id CODEX supports.
  • theme — optional. dark (default), light, or auto.
  • collapsed — optional. If present, renders a collapsed summary with click-to-expand.
  • show-ref — optional. If present, shows the reference label above the text.
  • link-to-codex — optional. If present, wraps in a link to the full CODEX app at the passage.

Theming via CSS variables

codex-reader {
  --cx-w-bg: #fff;
  --cx-w-fg: #1a1d28;
  --cx-w-accent: #0a6884;
  --cx-w-serif: "EB Garamond", Georgia, serif;
  --cx-w-mono: "JetBrains Mono", monospace;
  font-size: 17px;
}

Programmatic API

const el = document.createElement('codex-reader');
el.setAttribute('book', 'phil');
el.setAttribute('chapter', '2');
el.setAttribute('verse', '5-11');
document.body.appendChild(el);

// Listen to events
el.addEventListener('codex-loaded', e => console.log(e.detail));
el.addEventListener('codex-error',  e => console.warn(e.detail));
